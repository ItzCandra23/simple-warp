# Simple-Warp
Warp plugin for BDSX.

# Commands
* /warp - Teleport to warp.
* /warpui - Open warp ui.
* /addwarp - Create a warp.
* /removewarp - Remove a warp.
* /setwarp - Change a warp position and dimension.
